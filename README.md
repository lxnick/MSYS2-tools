# A Collection of tools built on MSYS2
用 MinGW 在 Windows 上製作的工具程式

## Setup MinGW/MSYS2  
[Setup MinGW/MSYS2](./setup-msys2.md)  

## 用法
1. 啟動 MinGW/MSYS2 環境
2. 進入 工作目錄
```
cd /c/Project/MSYS2-tools/<work-folder>
```
3. make

# Tool List
* DataCenterCsv  
用來過濾/提取 Data Center (USB) 存檔 ( export 到 CSV) 的資訊
